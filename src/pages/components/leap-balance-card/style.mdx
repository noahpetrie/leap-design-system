## Colour

| Element          | Property         | Colour token        |
| ---------------- | ---------------- | ------------------- |
| Leave type label | text-color       | `$text-primary`     |
| Balance text     | text-color       | `$text-primary`     |
| Unit indicator   | text-color       | `$text-secondary`   |
| Container        | background-color | `$layer-01`         |
| Border           | border-color     | `$border-subtle-01` |
| Progress track   | background-color | `$border-subtle-01` |
| Progress fill    | background-color | `$support-success`  |

### Low balance state

| Element        | Property         | Colour token       |
| -------------- | ---------------- | ------------------ |
