## Colour

### Toggle states

| State       | Element | Property         | Value       |
| ----------- | ------- | ---------------- | ----------- |
| Off         | Track   | background-color | `$toggle-off` |
| On          | Track   | background-color | `#0C8C5E` (Leap Green 500) |
| On + Hover  | Track   | background-color | `#096B48` (Leap Green 700) |
| Focus       | Track   | outline-color    | `#0C8C5E` (Leap Green 500) |
| Disabled    | Track   | background-color | `$text-disabled` |

### Label colours

| Element     | Property   | Colour token     |
| ----------- | ---------- | ---------------- |
| Label text  | text-color | `$text-primary`  |
| State text  | text-color | `$text-secondary` |
