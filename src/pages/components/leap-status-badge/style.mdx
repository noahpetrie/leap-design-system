## Colour

### Status colours

| Status    | Background colour    | Text colour          |
| --------- | -------------------- | -------------------- |
| Approved  | `$support-success`   | `$text-on-color`     |
| Denied    | `$support-error`     | `$text-on-color`     |
| Pending   | `$support-warning`   | `$text-primary`      |
| Cancelled | `$text-secondary`    | `$text-on-color`     |
| Active    | `$support-info`      | `$text-on-color`     |

### Common elements

| Element     | Property         | Colour token        |
| ----------- | ---------------- | ------------------- |
| Label text  | text-color       | See status table    |
| Container   | background-color | See status table    |

## Typography

| Element    | Font-size (md) | Font-size (sm) | Font-weight | Type token            |
| ---------- | -------------- | -------------- | ----------- | --------------------- |
| Label text | 12px           | 11px           | Semi-Bold   | `$label-01`           |
