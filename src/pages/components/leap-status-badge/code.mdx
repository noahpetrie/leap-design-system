## Getting started

To install `@carbon/react`, see the
[getting started guide](/developing/frameworks/react/).

```bash
npm install @carbon/react
```

## Basic usage

```jsx
import { LeapStatusBadge } from '@carbon/react';

function RequestHeader() {
  return <LeapStatusBadge status="approved" />;
}
```

## Small size

```jsx
<LeapStatusBadge status="pending" size="sm" />
```

## All statuses

```jsx
<div style={{ display: 'flex', gap: '0.5rem' }}>
  <LeapStatusBadge status="approved" />
  <LeapStatusBadge status="denied" />
  <LeapStatusBadge status="pending" />
  <LeapStatusBadge status="cancelled" />
  <LeapStatusBadge status="active" />
</div>
```

## Props

| Prop | Type | Required | Default | Description |
| ---- | ---- | -------- | ------- | ----------- |
| `status` | `"approved" \| "denied" \| "pending" \| "cancelled" \| "active"` | Yes | — | The status to display |
| `size` | `"sm" \| "md"` | No | `"md"` | Badge size |
