## Colour

| Element          | Property         | Colour token        |
| ---------------- | ---------------- | ------------------- |
| Employee name    | text-color       | `$text-primary`     |
| Role             | text-color       | `$text-secondary`   |
| Department       | text-color       | `$text-secondary`   |
| Balance badge    | text-color       | `$text-on-color`    |
| Balance badge    | background-color | `$support-success`  |
| Container        | background-color | `$layer-01`         |
| Border           | border-color     | `$border-subtle-01` |
| Avatar fallback  | background-color | `$layer-accent-01`  |
| Avatar fallback  | text-color       | `$text-on-color`    |

## Typography

| Element       | Font-size | Font-weight | Type token            |
| ------------- | --------- | ----------- | --------------------- |
| Employee name | 14px      | Semi-Bold   | `$heading-compact-01` |
| Role          | 12px      | Regular     | `$label-01`           |
| Department    | 12px      | Regular     | `$label-01`           |
| Balance badge | 12px      | Semi-Bold   | `$label-01`           |
